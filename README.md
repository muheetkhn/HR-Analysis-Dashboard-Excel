# 📊 HR Analytics Dashboard (Excel)

## 📌 Overview  
This project is an interactive HR Analytics Dashboard built using Microsoft Excel. It transforms raw employee data into meaningful insights to support data-driven decision-making in HR operations.

The dashboard helps analyze workforce trends such as employee distribution, attrition, performance, and salary structure in a clear and visually appealing format.

## 🎯 Objectives  
- Analyze employee data efficiently  
- Track attrition and workforce trends  
- Compare department-wise performance  
- Present insights through a clean dashboard  
- Enable quick decision-making using visuals  

## 📊 Key Insights  
- Employee distribution across departments  
- Attrition rate analysis  
- Department-wise performance comparison  
- Salary and demographic breakdown  
- Workforce trends and patterns  

## 🛠️ Tools & Techniques Used  
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning & Transformation  
- Excel Formulas & Functions  
- Dashboard Design Principles  

## ✨ Features  
- Interactive and dynamic dashboard  
- User-friendly layout  
- Clear data visualization  
- Easy to understand insights  
- Structured and clean design  

## 📁 Dataset  
The dataset includes employee-related information such as:
- Employee ID  
- Department  
- Salary  
- Job Role  
- Attrition  
- Demographics  

## 🚀 How to Use  
1. Download the Excel file  
2. Open in Microsoft Excel  
3. Go to the Dashboard sheet  
4. Use filters/slicers (if available) to explore insights
